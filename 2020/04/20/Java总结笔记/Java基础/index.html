Java基础

数据类型

Java是强类型语言：变量要先定义，后使用

数据类型分为两种：

  • 基本类型(primitive type)
  • 引用类型(reference type)(除了基本类型外，其余的都是引用类型)

八大基本数据类型：

整数类型 浮点类型 字符类型 布尔类型
byte float(F) char boolean
short double
int eg:float PI=3.14F;
long(L) eg:long L=19216811L;

注：除布尔类型外，其余类型都是数值类型

定义String类型的时候，要用双引号，不能是单引号

定义char类型的时候，要用单引号，不能是双引号

二进制 八进制 十进制 十六进制
0b 0 0x

强制类型转换在高－＞低的时候用

自动转换在低－＞高的时候自动使用

强制类型转换不能对布尔值进行转换

变量

类变量：在类的内部定义的变量，前加static，如果要转换为常量，可在static前或后加final。因为final是修饰符，所以加在哪里都可以，不能是double后面，因为double是数据类型，后应该接变量名称，所以不能加在double后。

实例变量：若不进行初始化，那么变量为默认值，0、0.0、false(boolean)、null(引用类型)

局部变量：不定义无法使用

常量：定义后不能修改，修饰符final

变量的命名规范

  • 所有变量、方法、类名：见其名，知其意
  • 类成员变量：首字母小写和驼峰原则(monthSalary)
  • 局部变量：首字母小写和驼峰原则
  • 常量：全大写，中间可加下划线
  • 类名：首字母大写和驼峰原则
  • 方法名：首字母小写和驼峰原则

整数运算时：

  • 有long类型参与的运算，结果为long类型

  • 没有long类型，都为int类型，无论有没有int参与
