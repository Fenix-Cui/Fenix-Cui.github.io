Java的三大版本

Write Once, Run Anywhere!

  1. JavaSE：标准版(桌面程序，控制台开发等)
  2. JavaME：嵌入式开发(手机，家电等)(卒，有更好的替代)
  3. JavaEE：企业级开发(web端，服务器开发等)

JDK、JRE、JVM

  • JDK：Java Development Kit

  • JRE：Java Runtime Environment

  • JVM：Java Virtual Machine

JDK > JRE > JVM

编译型和解释型

编译型语言：在执行程序前先编译，运行时使用编译的结果。程序执行效率高，依赖编译器，跨平台性差。如C、C++、Delphi等

解释型语言：在执行程序时编译，每执行一次都要翻译一次。程序执行效率低，依赖解释器，跨平台性号。如Python/JavaScript / Perl /Shell等。

Java程序运行机制

集成开发环境

集成开发环境（IDE，Integrated Development Environment ）是用于提供程序开发环境的应用程序，一般包括代码编辑器、编译器、调试器和图形用户界面等工具。集成了代码编写功能、分析功能、编译功能、调试功能等一体化的开发软件服务套。所有具备这一特性的软件或者软件套（组）都可以叫集成开发环境。(来自百度百科)
