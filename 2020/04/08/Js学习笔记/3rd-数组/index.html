3rd.数组

  1. 当给数组新增元素时，若超出当前数组范围，则会将数组长度扩大至相应长度，其中未赋值的位置为undefined。
  2. 减小arr.length的数值，会删除结尾的多余元素，将数组缩。
  3. 如果数组下标不连续，则成为稀疏数组。

数组API函数

数组转字符串

String(arr): 将arr中每个元素转为字符串，用逗号分割
固定套路:对数组拍照：用于鉴别数组是否被修改过
arr.join(“连接符”): 将arr中每个元素转为字符串，用自定义的连接符分割

拼接和选取

不直接修改原数组，而返回新数组！
拼接：arr.concat(value1, value2, value3, ...)
选取：slice()返回选优数组的一个子数组

选取arr中starti位置开始，到endi结束的所有元素组成新数组返回，原数组保持不变。
强调：凡是两个参数都是下标的函数，都有一个特性：含头不含尾。
如果选取的元素离结尾近，可以使用倒数下标。
复制数组：arr.slice(0, arr.length);
可以简写为：arr.slice();

修改数组

删除

删除：splice直接修改原数组
arr.splice(starti, n);
删除arr中starti位置开始的n个元素
其实：var deletes = arr.splice(starti, n);
返回值deletes保存了被删除的元素组成的临时数组

插入

arr.splice(starti, 0, value1, value2, ...);
在arr中starti位置，插入新的值1，2，…，原starti位置的值及其后面的值被向后顺位移动

替换

arr.splice(starti, n, value1, value2, ...);
先删除arr中starti位置的n个值，再在starti位置插入新值
注：删除的元素个数和插入的新元素的个数不必一致

颠倒数组

颠倒：reverse()颠倒数组中元素的顺序
arr.reverse();
仅支持原样颠倒数组，不负责排序

排序

排序API：arr.sort()
默认将所有元素转为字符串再排列
问题：只能排列字符串类型的元素
解决：使用自定义比较器函数
