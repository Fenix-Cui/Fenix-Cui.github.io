Java中碰到的坑(保持更新)

2020-4-21：

出现的问题：Occurrences found in comments, strings and non-code files (1 occurrence in 1 file) 1 usage。本来在src文件夹内的class移动不进base文件夹。

问题的原因：之前将Class文件重命名(Rename)了

解决的方法：把Search的那两个勾去掉就OK了，我只去掉了第一个就能移动了，第二个没有去掉

Search in comments and strings：在注释和字符串中搜索

Search for text occurrences：搜索出现的文本

2020-4-22：

出现的问题：IDEA包的结构显示不是树形结构的

问题的原因：！

解决的方法：点左面栏右上角的小齿轮，把FlattenPackage和Compact Middle Packages都勾选掉就OK了

Flatten Packages：扁平化包
Compact Empty Middle Packages：压缩空中间包(fanyi.baidu.com)

2020-4-25：

出现的问题：在给类的属性赋值的时候，打印不出来赋值的结果，语句没有错误

问题的原因：WDNMD

解决的方法：重启idea

2020-7-2:

出现的问题：Can’t create/write to file

问题的原因：my.ini中datadir错误

解决的办法：填写正确的datadir

![image-20200702165901371](D:\Fenix\blog\source_posts\Java中碰到的坑-保持更新\Can’t create-write to file.png)

2020-7-31:

出现的问题：有时快捷键alt+enter不能快速创建对象

问题的原因：无

解决的办法：使用ctrl+alt+v快捷键

2020-8-16：

出现的问题：添加新maven-webapp项目时，pom文件不自动添加parent，webapps文件夹没有标识，无法正确添加parent中的maven依赖

问题的原因：删除module后又重新创建一个与其相同名字的module(目前理解是这个原因)

解决的办法：删除.idea文件夹，重新加载后正常

2020-8-19：

出现的问题：JSP中添加图片不显示，路径正常

问题的原因：Tomcat部署了项目路径

解决的办法：JSP路径添加项目路径，在Tomcat中查看
